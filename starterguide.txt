# Car Customer Connect - Important Information Document
*Version 1.0 - MVP Specification*

## Product Overview

**Product Name:** Car Customer Connect  
**Product Type:** Micro-SaaS for Independent Auto Dealerships  
**Core Purpose:** Transform vehicle lifecycle events into automated social media marketing content  
**Target Users:** Independent auto dealership staff (sales managers, lot attendants, detailers, salespeople)

## Core MVP Features

### 1. Vehicle Management
- **VIN Scanner Integration**: Mobile camera-based VIN scanning with auto-population of vehicle details
- **Manual Entry Option**: Fallback for damaged VINs or quick additions
- **Bulk Import**: CSV upload for initial inventory setup
- **Smart Search**: Filter by status, make, model, stock number
- **Vehicle Status Tracking**: Acquisition → In Service → Ready for Sale → Sold

### 2. Content Generation System
- **AI-Powered Captions**: Context-aware post generation based on:
  - Vehicle details (make, model, year, features)
  - Event type (new arrival, ready for sale, sold)
  - Dealership voice/tone settings
  - Additional notes from staff
- **Photo Guidelines**: On-screen overlays showing optimal angles
- **Caption Variations**: Generate multiple options to choose from
- **Hashtag Intelligence**: Auto-suggest relevant, trending hashtags

### 3. Social Media Integration
- **Direct Posting**: One-click posting to connected platforms
- **Platform Optimization**: Adjust content for each platform's best practices
- **Scheduling Option**: Post immediately or schedule for optimal times
- **Multi-Account Support**: Manage multiple dealership locations

### 4. User Experience Enhancements
- **Mobile-First Design**: Optimized for lot staff using phones
- **Quick Actions**: Common tasks accessible in 1-2 taps
- **Voice-to-Text**: Add notes hands-free while inspecting vehicles
- **Offline Mode**: Capture photos/data, sync when connected

### 5. Analytics & Reporting
- **Post Performance**: Track engagement metrics
- **Inventory Turnover**: Time from acquisition to sale
- **Staff Activity**: Who's creating the most content
- **ROI Dashboard**: Connect social engagement to sales

## Additional Features (Not in Original Spec)

### 6. Customer Engagement Tools
- **Review Request Automation**: Send review invites post-sale
- **Birthday/Anniversary Reminders**: Re-engage past customers
- **Service Reminder Posts**: Automated maintenance reminders

### 7. Compliance & Legal
- **Photo Release Management**: Digital consent forms for customer photos
- **Disclosure Templates**: Ensure legal compliance in posts
- **Watermark Options**: Add dealership branding to photos

### 8. Team Collaboration
- **Role-Based Access**: Sales, service, management permissions
- **Approval Workflows**: Manager review before posting
- **Internal Notes**: Communication between team members

### 9. Inventory Showcase
- **Virtual Showroom**: Auto-generated webpage with current inventory
- **QR Code Generation**: For window stickers linking to vehicle details
- **Video Tour Templates**: Guide staff through video walkarounds

## Technical Architecture

### Frontend
- **Framework**: React with TypeScript for type safety
- **UI Library**: Tailwind CSS for rapid, consistent styling
- **State Management**: Zustand for simple, scalable state
- **Image Handling**: Compression and optimization before upload

### Backend
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth with role-based access
- **File Storage**: Supabase Storage for photos/videos
- **API Structure**: RESTful endpoints with real-time subscriptions

### Integrations
- **Social Platforms**: Facebook Graph API, Instagram Basic Display API
- **AI Service**: OpenAI GPT-4 for content generation
- **VIN Decoder**: NHTSA API for vehicle details
- **Analytics**: PostHog for product analytics

## Key Design Principles

1. **Speed First**: Every action optimized for speed on the lot
2. **Mobile Native**: Assume 80% usage on phones
3. **Foolproof**: Prevent errors, guide correct usage
4. **Instant Value**: Show results immediately
5. **Gradual Complexity**: Simple start, advanced features discoverable

## Technical Constraints & Considerations

- **API Rate Limits**: Queue system for social platform limits
- **Image Storage**: Optimize for cost with automatic cleanup
- **Offline Capability**: Critical for poor lot connectivity
- **Multi-Tenancy**: Proper data isolation between dealerships
- **Scalability**: Architecture supporting 1000+ dealerships
